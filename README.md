

# Потужний 4-канальний контролер освітлення

Контролер призначений для керування чотирма незалежними каналами освітлення за допомогою ШІМ. Підходить для амбіентного підсвічування та роботи з різними сенсорами присутності й освітленості.

## Основні можливості
- Керування **4 незалежними каналами**
- Регулювання яскравості через **ШІМ**
- Інтеграція з зовнішніми сенсорами:
  - Доплерівські сенсори присутності
  - PIR сенсори руху
  - Датчики освітленості
  - Інші сумісні датчики

## Система захисту
- **Температурний контроль** силових транзисторів
- Підтримка **зовнішнього NTC** біля навантаження
- **Захист по струму**
- **Контроль вхідної напруги** з блокуванням роботи в небезпечних діапазонах

## Технічні характеристики
- **Напруга живлення:** 12–26 В  
- **Споживання контролера:** ≤ 50 mA  
- **Максимальний струм комутації:**
  - до **12 А на канал**, або
  - до **12 А сумарно**

## Індикація роботи

### Зелений індикатор
- **Світиться** — нормальна робота  
- **Блимає** — перехідні процеси (зміна режиму, встановлення нового ШІМ)  
- **Вимкнений** — нештатна ситуація

### Червоний індикатор
- **Блимає** — відновлювані критичні стани  
  (критична напруга, перегрів, обмеження по струму, таймер блокування).  
  Виходи **вимкнені**  
- **Світиться** — фатальна помилка  
  (критичний перегрів, неможливість контролювати/відключити навантаження).  
  Скидання після перезапуску, якщо проблема не повторюється

## Програмування сценаріїв
Поведінку контролера, сценарії роботи та інтеграцію з сенсорами можна налаштовувати індивідуально під проєкт.
